# Big Data Analytics on Superstore Sales Dataset using Hadoop Ecosystem

This project demonstrates the implementation of Big Data Analytics using various Hadoop ecosystem technologies, including:

- Hadoop HDFS
- MapReduce
- Apache Pig
- Apache Hive
- Apache Spark (PySpark)

The project analyzes the Superstore Sales dataset to generate business insights such as category-wise sales, region-wise sales, category-wise profit, region-wise profit, and advanced analytics using Spark.
